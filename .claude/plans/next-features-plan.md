## 추가 후보 (우선순위순)

### A. PDF 도구 (tools) — 사용자 수요 매우 높음

#### 1. PDF 도구 모음 (`/pdf-tools`)
- **컴포넌트**: `PdfTools.tsx`
- **기능**:
  - PDF 합치기 (여러 파일 → 하나)
  - PDF 분할 (페이지별 분리)
  - PDF → 이미지 변환
  - 이미지 → PDF 변환
  - PDF 페이지 회전
- **기술**: pdf-lib (합치기/분할), jsPDF (이미 설치됨)
- **난이도**: 중상
- **의존성**: `npm install pdf-lib` 필요

## 구현 시 참고사항

### 패키지 설치 필요 여부
- **설치 불필요**: 테트리스, 숫자야구, 기억력, 스네이크, 대비체커, Favicon, OG미리보기, 이미지합치기, EXIF(직접파싱), 텍스트정리
- **설치 필요**: PDF 도구(`pdf-lib`), GIF 만들기(`gifenc`)

### 병렬 에이전트 활용 전략
- 게임끼리 병렬 (테트리스 + 숫자야구 + 기억력)
- 개발자 도구끼리 병렬 (대비체커 + Favicon)
- 번역은 컴포넌트와 같은 에이전트가 함께 작성 (키 불일치 방지)
- 공유 파일(menuConfig, ko.json, en.json, sitemap.ts)은 메인 에이전트가 일괄 수정

### Tesseract.js v6 참고
- `recognize()` 반환 타입 `Page`에서 words 접근: `data.blocks[].paragraphs[].lines[].words[]`
- `data.words` 직접 접근 불가 (빌드 에러)
- `createWorker(langs)` + dynamic import 패턴 사용
